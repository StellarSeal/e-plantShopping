# Paradise Nursery

Paradise Nursery is a React front-end shopping application for browsing houseplants, adding plants to a Redux-powered shopping cart, updating quantities, and reviewing cart totals before checkout.

## Project Pages

- Landing page with the Paradise Nursery company name, background image, company description, and Get Started button.
- Product listing page with categorized houseplants, product thumbnails, prices, and Add to Cart buttons.
- Shopping cart page with item quantities, item totals, total cart quantity, total cart amount, delete controls, continue shopping, and checkout messaging.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
